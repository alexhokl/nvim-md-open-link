# nvim-md-open-link
Neovim plugin to open links in markdown files

## Installation

### Lazy.nvim

```lua
{
    "alexhokl/nvim-md-open-link",
    opts = {
        keymap = "gb",
    },
},
```

## Tests

This project uses `busted` for unit tests.

Install test dependencies and run:

```bash
task setup
task test
```
